Linux zgrep使用笔记

Posted on 2017-03-17 | In Linux | | Visitors

grep 可以用来搜索文件内容,而基于grep的 zgrep这可以用来在压缩包中搜索内容而不需要事先解压文件.

  • 注意需要加上 -a 参数,让其二进制文件当做文本处理
  • tar -u -r 先压缩包中添加文件
  • tar -H 显示文件名
  • tar -n 显示行号
